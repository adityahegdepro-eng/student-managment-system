# Student Management System

## Project Description
A simple Python-based student management system that helps track:

- Student names
- Attendance percentage
- Assignment status

This project is built for learning:
- Python
- Logic building
- Project structure
- GitHub workflow

---

## Features

- Add student details
- Calculate attendance percentage
- View assignment status

---

## Technologies Used

- Python
- VS Code
- Git & GitHub
